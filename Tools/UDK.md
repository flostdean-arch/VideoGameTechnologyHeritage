# UDK(Unreal Engine 3)への現在のアクセス手段

- 対象: Unreal Development Kit(UDK) = Unreal Engine 3の無料版
- 年代: 記録時点(2026年7月)の状況
- 難易度: 初級
- 出典: Epic Games公式サイト "Unreal Engine 3" ページ、Internet Archive `udkinstaller`、ModDB/IndieDB UDKダウンロードページ

## 背景

Epic Games公式サイトは、UE3およびUDKについて「更新提供を終了した。新規プロジェクトはUE4以降を利用してほしい」と明言しており、**公式からの新規ダウンロード手段は現在存在しない**。

詳細な経緯・技術解説は [`GameEngines/UnrealEngine/UnrealEngine3/README.md`](../GameEngines/UnrealEngine/UnrealEngine3/README.md) を参照。このページでは「今、実際にどこで入手できるか」だけを記録する。

## 現在確認できている入手経路(非公式ミラー)

**いずれもEpic公式の配布ではない、有志によるミラーである点に注意。将来的にこれらのミラー自体が消える可能性もあるため、定期的にリンク切れを確認すること。**

### Internet Archive

- アイテム名: `udkinstaller`
- 内容: 2009年11月〜2015年2月にリリースされた各バージョンのUDKインストーラー(ベータ版含む)、計27ファイル・約37GB
- 配布形式: Windows実行ファイル(.exe)、およびTorrent

### ModDB / IndieDB

- UDK専用のダウンロードページがあり、インストーラー本体に加えて、UDK/Unreal Engine 3のオフラインドキュメントも配布されている

## 今後この記事に追記したいこと

- [ ] 各バージョンのインストーラーが実際に現行OS(Windows 11等)で動作するかの検証
- [ ] 上記2経路以外のミラー(あれば)の調査
- [ ] リンク切れが発生していないかの定期確認(確認日を記録していく形式にする)
- [ ] UnrealEd(UDK同梱のエディタ)を使い始めるための最低限の手順
