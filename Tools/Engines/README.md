# Engines

## 概要

このフォルダでは、ゲームエンジンやゲーム開発環境に関する情報をまとめます。

対象はエンジンそのものの技術解説ではなく、
現在利用するための情報、導入方法、関連資料、保存状況などを扱います。

ゲームエンジンの内部構造や歴史的発展については、
以下の項目を参照してください。

- `EngineArchitecture/`
- `HistoricalTechnology/GameEngineHistory/`

---

## 主な対象

- 商用ゲームエンジン
- オープンソースゲームエンジン
- 過去の開発環境
- 配布終了したエンジン環境
- 復元・保存対象となる開発環境

---

## 記録内容

各エンジンについて以下の情報を記録します。

- 概要
- 対応プラットフォーム
- 現在の入手状況
- 関連資料
- 利用方法
- 保存上の注意点

---

## 目的

ゲーム開発技術を実際に確認・検証するためのエンジン環境情報を整理します。
