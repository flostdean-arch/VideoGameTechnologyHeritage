# SourcePorts

## 概要

このフォルダでは、過去のゲームエンジンやゲーム作品を現代環境で動作させるための
ソースポートプロジェクトについてまとめます。

ソースポートとは、既存ゲームのエンジン部分を解析・移植し、
現代のOSやハードウェア上で動作可能にするプロジェクトです。

---

## 主な対象

- 旧作ゲームエンジン移植
- オープンソース化されたゲームエンジン
- ファンによる復元プロジェクト
- 強化版エンジン

---

## 記録内容

各プロジェクトについて以下を記録します。

- 対応作品
- 元となるゲーム
- 対応プラットフォーム
- 必要なデータ
- 導入方法
- 技術的特徴

---

## 代表例

- EDuke32
- ioquake3
- RBDOOM-3-BFA

---

## 目的

過去のゲーム技術を保存し、
現代環境で再利用・研究できる状態にするための情報を記録します。
