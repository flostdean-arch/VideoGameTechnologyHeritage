## この項目について(暫定)

`GameEngines/`が「エンジンそのものの技術解説」なのに対し、`Tools/`は以下のような**実際に触るための手段**を記録する想定(用途は今後調整可)：

- 各エンジンを現在動かすための非公式ポート・復元プロジェクトの情報(例: EDuke32, ioquake3, RBDOOM3-BFA等)
- 保存・アーカイブ作業自体に使えるツール(逆コンパイラ、アセット抽出ツール等)
- このリポジトリの執筆・翻訳作業を助けるツール
