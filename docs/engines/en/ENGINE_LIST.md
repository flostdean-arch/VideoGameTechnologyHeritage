# Engine List (Candidates)

This file serves as a starting point for the VideoGameTechnologyHeritage repository, summarizing "old game engines" that are candidates for preservation or reproduction, with short status notes in English.

Format (per engine):
- Engine name
- Summary
- How to obtain (official / community / archive)
- Representative titles (examples)
- Major source ports / tools
- License / rights notes
- Priority (High / Medium / Low)

---

1) Unreal Engine 1
- Summary: Early Unreal engine. A rich modding culture grew around tools such as UnrealEd.
- How to obtain: Old distributions (Unreal Gold etc.), community resources; official source releases are limited.
- Representative titles: Unreal, early Unreal Tournament
- Major tools / source ports: UnrealEd, unofficial docs/tutorials
- License: Contains commercial assets; distribution requires caution. Verify handling of tools and docs.
- Priority: High

2) Unreal Engine 2
- Summary: Successor to UE1. Modding techniques continued but documentation is scattered.
- How to obtain: Game-packaged SDKs and community archives
- Representative titles: Unreal Championship 2, etc.
- Major tools / source ports: UnrealEd (version-dependent)
- License: Check licenses for binaries and assets
- Priority: Medium

3) id Tech 1 (Doom family)
- Summary: Engine used by Doom. Source code has been released and many source ports exist.
- How to obtain: Official source releases, community projects (GZDoom, etc.)
- Representative titles: Doom, Doom II
- Major source ports / tools: GZDoom, Chocolate Doom
- License: Follow the source license (historically mixed, not always GPL).
- Priority: Medium

4) id Tech 2–4 (Quake family to Doom 3)
- Summary: Quake lineage (Quake, Quake II, Quake III) and id Tech 4 (Doom 3). Many are source-released and have active ports.
- How to obtain: Source releases, community builds, source ports
- Representative titles: Quake series, Doom 3
- Major source ports / tools: DarkPlaces, ioquake3, vkQuake, ioquake2, various ports
- License: Check each project's license (GPL, etc.)
- Priority: Medium

5) Build Engine
- Summary: Engine used by Duke Nukem 3D and others. EDuke32 and Raze enable running on modern systems.
- How to obtain: Source ports (EDuke32, etc.)
- Representative titles: Duke Nukem 3D, Shadow Warrior
- Major source ports / tools: EDuke32, Raze
- License: Verify source port licenses
- Priority: Medium

6) GoldSrc / Source
- Summary: Valve's GoldSrc (early Half-Life) and Source (Half-Life 2). SDKs and community docs are plentiful, making modding accessible.
- How to obtain: Via Steam and SDKs, community documentation
- Representative titles: Half-Life series, Counter-Strike
- Major tools / source ports: Source SDK, Hammer Editor, various community tools
- License: Game assets are commercial; check SDK terms of use
- Priority: High

7) SCUMM family (LucasArts adventure engines)
- Summary: 2D adventure engine family. High fidelity preservation via ScummVM.
- How to obtain: ScummVM project; original game data requires ownership
- Representative titles: Monkey Island, Day of the Tentacle
- Major source ports / tools: ScummVM
- License: Game data depends on rightsholders. Engine compatibility implementations are OSS.
- Priority: Low–Medium

8) RenderWare
- Summary: Commercial middleware from the early 2000s. Obtaining and licensing can be complex.
- How to obtain: Archives and community materials (difficult to obtain)
- Representative titles: Many early PS2-era titles
- Major source ports / tools: Community analyses and docs
- License: Commercial licensing considerations — be careful
- Priority: Low

9) LithTech
- Summary: Monolith's engine. Documentation is limited.
- How to obtain: Archives, community resources
- Representative titles: No One Lives Forever, etc.
- Major source ports / tools: Community analysis
- License: Commercial and complex
- Priority: Low

10) Early CryEngine / other older commercial engines
- Summary: Older versions of CryEngine and other commercial engines have scattered documentation.
- How to obtain: Rare official archives, community resources
- Representative titles: Early Far Cry, etc.
- Major source ports / tools: Limited
- License: Be cautious
- Priority: Low
