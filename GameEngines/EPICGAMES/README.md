# Epic Games

## 概要

Epic Games（旧：Epic MegaGames）は、アメリカのゲーム開発会社です。

1998年に発売された『Unreal』をきっかけに、自社開発エンジンである**Unreal Engine**シリーズを展開し、現在では世界で最も普及しているゲームエンジンの一つとなっています。

本リポジトリでは、ゲーム作品よりも**ゲームエンジンの技術的発展**に着目し、その歴史や設計思想、現在の研究環境を記録します。

# 技術的な特徴

Unreal Engineシリーズでは、時代ごとにゲーム開発を大きく変える技術が導入されました。

* UnrealScript
* UnrealEd
* BSPベースのレベルデザイン
* Kismet
* Blueprint
* Nanite
* Lumen

これらの技術は現在のゲームエンジンにも大きな影響を与えています。
