# Unreal Engine 2

## 概要

Unreal Engine 2（UE2）は、2002年に登場したUnreal Engineシリーズ第2世代です。

DirectX 8世代のGPU性能向上に対応し、大規模なマップやStatic Meshなど、現代のゲームエンジンへ繋がる技術を数多く導入しました。

---

# 主な特徴

- Static Mesh
- Karma Physics
- DirectX 8対応
- UnrealScriptの拡張
- UnrealEd 3
