# Unreal Engine 3

## 概要

Unreal Engine 3（UE3）は、2006年より本格的に採用されたDirectX 9世代のゲームエンジンです。

Kismet、Lightmass、マルチスレッド対応など、多くの革新的な技術を導入し、現在のゲームエンジンにも大きな影響を与えました。

---

# 主な特徴

- DirectX 9
- マルチスレッド
- Kismet
- Lightmass
- UnrealScript
