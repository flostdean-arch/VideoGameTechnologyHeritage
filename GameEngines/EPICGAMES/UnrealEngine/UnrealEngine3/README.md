# Unreal Engine 3

## 概要

Unreal Engine 3（UE3）は、Epic Gamesによって開発されたDirectX 9世代のゲームエンジンです。

2006年頃から本格的に採用され、PlayStation 3、Xbox 360世代のHDゲーム開発を代表するエンジンとなりました。

UnrealScript、Kismet、Lightmass、UnrealEdなど、多くの技術を導入し、現在のゲームエンジン設計にも大きな影響を与えています。

---

# 開発背景

UE3は、HD解像度・次世代家庭用ゲーム機時代への対応を目的として開発されました。

UE2世代までの設計を発展させながら、

- 高品質なリアルタイムレンダリング
- 大規模ゲーム開発
- 複数プラットフォーム対応

を実現するための基盤として設計されています。

# 現在の研究環境

★★★☆☆

公式配布は終了しています。

UE3本体およびUDKは現在公式には提供されていません。

一方で、過去の開発資料やコミュニティによる保存活動により、技術研究は一部可能です。

---

# 技術的影響

UE3は、現代のゲームエンジン設計へ多くの影響を与えました。

特に、

- Kismet → Blueprint
- UnrealScript → UE4 C++ / Blueprint
- Lightmass → Lumen
- UnrealEd → Unreal Editor

など、多くの思想が後継世代へ継承されています。
