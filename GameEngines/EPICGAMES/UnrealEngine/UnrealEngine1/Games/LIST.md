# Unreal Engine 1 Game List

## 概要

本ドキュメントでは、**Unreal Engine 1** を採用した主なゲームタイトルを一覧形式でまとめる。

Unreal Engine 1はEpic Gamesだけでなく、多くのサードパーティ企業へライセンス提供され、FPS、アクション、RPGなど幅広いジャンルの作品で採用された。作品ごとにエンジンへ独自の改良が施されており、Unreal Engineの技術的発展を知る上でも重要な資料となっている。

詳細な情報については、各タイトルの個別ドキュメントを参照。

# 掲載基準

本一覧には、以下の条件を満たす作品を掲載する。

- Unreal Engine 1を採用した商用タイトル
- Unreal Engine 1をベースに開発・拡張された作品
- 技術史的に重要なライセンスタイトル

リマスター版や移植版については、Unreal Engine 1を使用している場合のみ掲載対象とする。
