# Games

## 概要

このディレクトリでは、**Unreal Engine 1** を採用したゲームタイトルについてまとめる。

Unreal Engine 1は1998年の『Unreal』で初めて採用され、その後もFPS、アクション、RPGなど幅広いジャンルの作品にライセンス提供された。採用作品ごとにエンジンは独自の拡張や改良が行われており、ゲームエンジンの進化を知る上でも重要な資料となっている。

本ディレクトリでは、各タイトルの概要だけでなく、Unreal Engine 1で利用・拡張された技術や、エンジンの発展に与えた影響についても整理する。

# 収録方針

本ディレクトリでは、以下の基準で資料を作成する。

- Unreal Engine 1を採用したタイトル
- Unreal Engine 1をベースに独自の改良・拡張が施されたタイトル
- Unreal Engine 1の技術的発展に影響を与えたタイトル
- 歴史的・技術的価値の高い作品

MODやユーザー制作コンテンツについては、原則として対象外とし、必要に応じて関連資料で扱う。
