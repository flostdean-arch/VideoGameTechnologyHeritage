# Crytek

## 概要

Crytekは、ドイツで設立されたゲーム開発会社です。

『Far Cry』や『Crysis』シリーズの開発を通じて、自社製ゲームエンジン **CryEngine** を発展させ、リアルタイムグラフィックス技術の進歩に大きな影響を与えました。

特に植生表現、広大な屋外マップ、HDRレンダリング、物理ベースレンダリング（PBR）などの技術は、その後のゲームエンジンにも大きな影響を与えています。

本リポジトリでは、ゲーム作品ではなく、CryEngineシリーズの技術的な進化と保存状況を中心に記録します。

# 技術的な特徴

CryEngineシリーズでは、当時としては最先端のリアルタイムグラフィックス技術が数多く導入されました。

代表的な技術として、

* 大規模な屋外レンダリング
* リアルタイム植生システム
* HDR（High Dynamic Range）レンダリング
* Deferred Rendering
* Physically Based Rendering（PBR）
* Sandbox Editor
* リアルタイム物理シミュレーション

などが挙げられます。

これらの技術は、現在のゲームエンジンにも広く受け継がれています。

---

# 現在の研究環境

CryEngineシリーズは、世代によって研究環境が大きく異なります。

初期世代（CryEngine 1・2）は、現在では開発環境やSDKの入手が難しくなっており、実機で検証するハードルが高くなっています。

一方、CryEngine 3以降はSDKや公式ドキュメントが公開されていた時期があり、比較的資料が残っています。

# 本リポジトリでの扱い

本リポジトリでは、CryEngineを**ゲームグラフィックス技術の歴史を語る上で欠かせないゲームエンジン**として位置付けています。

一方で、初期世代のSDKや開発環境は現在では入手が難しく、実際に研究できる環境は限られています。

そのため、技術解説に加え、各世代の保存状況や現在の研究環境についても記録していきます。
