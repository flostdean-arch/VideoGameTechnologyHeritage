# id Software

## 概要

id Softwareは、アメリカのゲーム開発会社です。

『Wolfenstein 3D』『DOOM』『Quake』シリーズを通じて、FPSというジャンルだけでなく、ゲームエンジン技術そのものの発展を牽引してきました。

現在でも、多くの世代のエンジンソースコードが公開されており、ゲームエンジン研究において極めて重要な存在です。
