# id Tech

## 概要

id Techは、id Softwareが開発してきたゲームエンジンシリーズです。

『DOOM』『Quake』シリーズをはじめとする数多くの作品で採用され、リアルタイム3Dグラフィックス、ネットワークゲーム、ゲームエンジンライセンスなど、ゲーム業界へ大きな影響を与えました。

また、多くの世代でソースコードが公開されており、現在でも研究・教育用途で利用できる代表的なゲームエンジンでもあります。
