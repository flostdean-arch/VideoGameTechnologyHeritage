# id Tech 1

## 概要

id Tech 1は、1993年発売の『DOOM』で採用されたゲームエンジンです。

現在では「DOOM Engine」と呼ばれることもあります。

BSPやWADファイル、2.5Dマップ構造など、多くのゲーム技術の基礎を築きました。
