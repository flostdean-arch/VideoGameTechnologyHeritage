# Valve

## 概要

Valveは、アメリカのゲーム開発会社です。

『Half-Life』シリーズの開発を通じて、GoldSrc、Source、Source 2といったゲームエンジンを発展させてきました。

特にGoldSrcはid Tech 2をベースとして独自に発展し、その後のSource Engineへと受け継がれています。

---

# エンジンファミリー

* GoldSrc
* Source
* Source 2

---

# 技術的な特徴

Valveのゲームエンジンは、

* 高度な物理演算
* マテリアルシステム
* SDKによるMOD開発
* Hammer Editor

などを特徴としています。

Half-Lifeシリーズだけでなく、多くのMOD文化やインディーゲーム開発にも大きな影響を与えました。
