# Unreal Engine 3 (UE3)

- 対象: Unreal Engine 3(Unreal Engine 2の後継)
- 年代: 2004年〜(2006年末より採用タイトルが発売開始)
- 難易度: 中級〜上級
- 出典: (執筆時に追記。Wikipedia "Unreal Engine 3" "UDK Licensing Resources" を参考に確認済みの範囲のみ記載)

## 概要

Unreal Engine 3は、Epic GamesがUnreal Engine 2の後継として開発したエンジン。開発自体は2004年頃(build 100)に始まり、これを採用した最初期のタイトル(Gears of Warなど)は2006年末に発売された。DirectX 9をベースラインとし、マルチスレッド対応をいち早く取り入れた世代の一つとされる。

## 現在の入手性・注意点

**このセクションはこのプロジェクトの読者にとって特に重要なので、必ず維持してください。**

- **エンジンのC++コア部分のソースコード**: 現在も非公開(License: Proprietary)。Epicから公式にGPL等でオープンソース化されたことはない
- **UDK(Unreal Development Kit)**: 2009年よりEpicが無料配布したツールキット。UnrealScriptおよびツール群へのアクセスは提供されるが、**C++コアのソースコードは含まれない**点に注意。商用利用には別途ライセンス契約が必要
- → UE1/UE2の「ゲーム本体は無料配布・ツールも同梱」というパターンとは異なり、UE3は「UDKという専用の無料ツールキットが別途提供された」という経緯である点を区別して記載すること
