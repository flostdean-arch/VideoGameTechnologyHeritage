# Unreal Engine 3 (UE3)

- **対象**: Unreal Engine 3（Unreal Engine 2の後継）
- **開発開始**: 2004年頃
- **採用タイトル発売**: 2006年末～
- **難易度**: 中級〜上級

---

# 概要

Unreal Engine 3（UE3）は、Epic GamesがUnreal Engine 2の後継として開発したゲームエンジンである。

開発は2004年頃（Build 100）から開始され、2006年発売の『Gears of War』などを皮切りに多くのゲームで採用された。

DirectX 9世代を代表するゲームエンジンの一つであり、マルチスレッド対応や高度なシェーダー技術など、後のゲームエンジンに大きな影響を与えた。

# 現在の研究環境（2026年時点）

## ソースコード

- C++コア部分は現在も非公開
- Epic Gamesによるオープンソース公開は行われていない

## UDK（Unreal Development Kit）

2009年よりEpic Gamesが無料公開していた開発キット。

利用できるもの

- UnrealScript
- UnrealEd
- Kismet
- コンテンツ制作ツール

利用できないもの

- エンジンC++ソースコード

※ 商用利用にはライセンス契約が必要だった。

## 現在の公式サポート

Epic GamesによるUDKの配布・更新は終了している。

現在はUE4以降の利用が案内されており、公式から新規にUDKを入手する手段は存在しない。

詳細な入手方法については **Tools/UDK.md** を参照。

# 後継技術への影響

UE3で採用された多くの設計思想は、後のUnreal Engine 4・5にも引き継がれている。

例：

- Kismet → Blueprint
- Content Browserの設計
- Lightmass
- アセット管理手法
- エディタ中心のワークフロー

これらの詳細については Unreal Engine 4 の記事でも扱う予定。

# 考察

UE1・UE2についてはOldUnrealによる継続的な保守・配布体制が存在する。

一方、UE3は公式による配布終了後、コミュニティによる保存活動に大きく依存する状況となっている。

そのため、本リポジトリではUE3を「保存・記録の重要性が高いゲームエンジン」の一つとして位置付けている。
