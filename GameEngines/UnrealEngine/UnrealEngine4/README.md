# Unreal Engine 4

## 概要

Unreal Engine 4（UE4）は、2014年に公開されたゲームエンジンです。

BlueprintやPhysically Based Rendering（PBR）の導入により、多くのインディーゲームやAAAタイトルで採用されました。

GitHubを通じてソースコードが公開された最初のUnreal Engineでもあります。
