# Unreal Engine 1

## 概要

Unreal Engine 1（UE1）は、Epic Games（当時 Epic MegaGames）が開発したシリーズ最初のゲームエンジンです。

1998年発売の『Unreal』のために開発され、その後『Unreal Tournament』『Deus Ex』など数多くの作品で採用されました。

シリーズの基礎となる多くの設計思想は、この世代で確立されています。

---

# 主な特徴

- UnrealScriptによるゲームロジック
- UnrealEdによるレベル制作
- BSPを利用した空間構築
- ソフトウェア・Glide・Direct3D・OpenGLへの対応
- Client-Server方式のネットワーク
