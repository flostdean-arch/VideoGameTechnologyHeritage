# 3D Realms

## 概要

3D Realmsは、1990年代のPCゲーム市場を代表するゲーム開発会社・パブリッシャーです。

『Duke Nukem 3D』をはじめとする作品で採用された**Build Engine**は、独特な2.5D表現や高速なレンダリングにより、多くのゲームへ採用されました。

本リポジトリでは、Build Engineの技術的特徴や保存状況を中心に扱います。

# 技術的な特徴

Build Engineでは、

* Sectorベースのマップ構造
* Portalによる可視判定
* Room over Room（TROR）
* CONスクリプト
* Build Editor

など、独自性の高い技術が採用されました。

---

# 本リポジトリでの扱い

Build Engineには公式な世代区分が存在しません。

そのため、本リポジトリでは技術ごとに記事を整理し、Build Engine全体の発展や、現代のソースポートによる保存活動についても記録します。
