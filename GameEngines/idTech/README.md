# id Tech

- 対象: id Techシリーズ(id Software)
- 年代: 1993年〜(Doomにて初採用、後年GPLで順次ソース公開)
- 出典: (執筆時に追記)

## この系統について

id Software(John Carmack)が開発するゲームエンジンシリーズ。Unreal Engineと異なり、初期の世代(id Tech 1〜4)は発売から数年〜十数年後にGPLライセンスでソースコードそのものが公開されているのが最大の特徴。「エンジンのC++コアが実際に読める」希少な系統として、このプロジェクトでは優先的に記録する。

このフォルダでは世代ごとに以下のように分けて記録する。個別の技術詳細は各世代フォルダを参照:

- [`idTech1/`](./idTech1/README.md) — Doomエンジン(1993年〜)
- [`idTech2/`](./idTech2/README.md) — Quakeエンジン(1996年〜)
- [`idTech3/`](./idTech3/README.md) — Quake IIIエンジン(1999年〜)
- [`idTech4/`](./idTech4/README.md) — Doom 3エンジン(2004年〜)

このREADMEには、系統全体の概要と世代間の進化の要点だけを記録し、個別の技術詳細は書かないこと。

## このシリーズ全体を通じた注意点

- id Tech 1〜4は「発売数年後にGPLで完全ソース公開」という共通パターンを持つが、id Tech 5以降は非公開(プロプライエタリ)のままである点に注意。このプロジェクトで扱うのは主にid Tech 1〜4の範囲
- ソースが公開されているとはいえ、当時のアセット(グラフィック・サウンド等)は別途著作権が生きている場合が多いため、技術解説では「エンジンのコード」と「ゲームのアセット」を明確に区別すること
