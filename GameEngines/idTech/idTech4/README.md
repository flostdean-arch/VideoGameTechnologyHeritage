# id Tech 4(Doom 3エンジン)

- 対象: id Tech 4(Doom 3エンジン)
- 年代: 2004年〜(『Doom 3』にて初採用)
- 難易度: 中級〜上級
- 出典: (執筆時に追記)

## 概要

id Softwareが2004年発売の『Doom 3』のために開発したエンジン。id Tech 3をベースに開発が始まったが、レンダラーを中心に大部分が書き直されている。開発言語がCからC++へ移行した世代でもある。統一的なライティング・シャドウ技術(いわゆる「Carmack's Reverse」によるステンシルシャドウボリューム)が特徴。『Quake 4』『Prey』『Wolfenstein』(2009年版)等にも採用された。

## 現在の入手性・注意点

**このセクションはこのプロジェクトの読者にとって特に重要なので、必ず維持してください。**

- **ソースコード**: John Carmackが複数年にわたり公開を予告し続けた末、2011年11月22日にGPL-3.0-or-laterでGitHub上に公開された。公開にあたり、特許問題(ステンシルシャドウボリューム関連)を避けるため一部コードが書き直されている
- **ゲームアセット**: エンジンのソースコードとは別に著作権が生きているため、技術解説ではエンジンのコードとアセットを明確に区別すること
- `The Dark Mod`(Doom 3ベースの独立作品)等、id Tech 4を土台にした活発な派生プロジェクトが存在する
