# id Tech 2(Quakeエンジン)

- 対象: id Tech 2(Quakeエンジン)

- 難易度: 中級
- 出典: (執筆時に追記)

## 概要

id Softwareが1996年発売の『Quake』のために開発したエンジン。id Tech 1の擬似3D(2.5D)表現から脱却し、真の3D空間としてレベルを構築できるようになった世代。『Quake II』にも派生版が使われている(Quake IIエンジンとして区別されることもある)。

このエンジンは公開後すぐに多数の派生("フォーク")を生み、後のid Tech 3やValveのGoldSrc(Half-Lifeのベースエンジン)にも影響を与えた。
