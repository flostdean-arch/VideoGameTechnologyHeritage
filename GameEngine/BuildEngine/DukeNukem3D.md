# Duke Nukem 3D

- 対象: Duke Nukem 3D (3D Realms / Apogee Software)
- 発売年: 1996年
- プラットフォーム: PC(DOS/Windows), PlayStation, Sega Saturn など
- エンジン: Build Engine

## 概要

『Duke Nukem 3D』は、Ken Silverman の Build Engine を採用した FPS ゲーム。1996年の PC ゲーム業界において、『Doom』『Quake』と並ぶ重要なタイトルとなった。

当時のゲームの中でも特に **キャラクターの個性的なボイス演技** と **インタラクティブなゲーム世界（破壊可能なオブジェクト、会話など）** が特徴的であり、単なるテクニカルな技術だけでなく、ゲームデザインの観点でも後続作品に大きな影響を与えた。

## 技術的な特徴

### グラフィックス

- **セクタベースの 3D レンダリング**
  - Build Engine は voxel ベースではなく、セクタ・ポーテルシステムを採用
  - id Tech 1(Doom)の 2.5D アプローチから、より自由度の高い 3D 空間を実現

- **テクスチャマッピング**
  - 当時としては高度なテクスチャの自由度
  - ウォールスプライト（敵キャラクター）の効果的な使用

### ゲームデザイン

- **破壊可能な環境**
  - オブジェクトの動的な破壊・変形
  - ゲーム世界への強いインタラクティビティ

- **ボイスアクティング**
  - キャラクターボイス（Duke の一人称視点での会話など）
  - 当時の技術レベルとしては珍しい音声表現

## ソースコード公開

『Duke Nukem 3D』のソースコードは **2003年4月1日** に 3D Realms によって公開された。詳細は [`SourceCode.md`](./SourceCode.md) を参照。

## 後続作品への影響

- **EDuke32** - 最も有名なソースポート。グラフィックスの近代化、拡張プレイ機能を追加
- **Duke Nukem Forever** - 後年の公式続編（ただし開発難航で有名）
- その他多数の MOD / クローン
