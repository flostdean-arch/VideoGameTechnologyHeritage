# id Tech 1(Doomエンジン)

- 対象: id Tech 1(Doomエンジン)
- 年代: 1993年〜(『Doom』にて初採用)
- 難易度: 中級
- 出典: (執筆時に追記)

## 概要

id Softwareが1993年発売の『Doom』のために開発したエンジン。『Doom II: Hell on Earth』のほか、Raven Softwareの『Heretic』『Hexen: Beyond Heretic』等、ライセンス先のタイトルにも採用された。

BSP(Binary Space Partitioning)を用いた擬似3D表現が特徴で、実際には2Dマップデータから3D風の視点描画を行う手法(いわゆる「2.5D」)を採用している。

## 現在の入手性・注意点

**このセクションはこのプロジェクトの読者にとって特に重要なので、必ず維持してください。**

- **ソースコード**: 1997年12月にid Softwareからプロプライエタリなライセンスで一度公開され、1999年10月にGPL-2.0-or-laterへと再ライセンスされた。現在もGitHub等で公式にソースを確認できる
- **『Heretic』『Hexen: Beyond Heretic』(Raven Software製)**: 2008年9月にGPL-2.0のみでソース公開されている
- **ゲームアセット(WADファイル等)**: エンジンのソースコードとは別に著作権が生きているため、技術解説ではエンジンのコードとアセットを明確に区別すること
- id Tech 1系は現在も`Freedoom`など活発な派生・互換実装が存在し、実際に動かしながら学べる環境が整っている
