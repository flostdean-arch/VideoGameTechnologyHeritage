# id Tech 3(Quake IIIエンジン)

- 対象: id Tech 3(Quake III Arenaエンジン)
- 年代: 1999年〜(『Quake III Arena』にて初採用)
- 難易度: 中級
- 出典: (執筆時に追記)

## 概要

id Softwareが1999年発売の『Quake III Arena』のために開発したエンジン。id Tech 2をベースにしつつ大部分が書き直されており、当時のUnreal Engineと市場で競合していた世代でもある。OpenGL対応グラフィックアクセラレータを必須とする点が特徴。

## 現在の入手性・注意点

**このセクションはこのプロジェクトの読者にとって特に重要なので、必ず維持してください。**

- **ソースコード**: 2005年のQuakeConでJohn Carmackが公開を発表し、同年8月19日にGPL-2.0-or-laterで公開された。現在もGitHub上で公式に確認できる
- **ゲームアセット**: エンジンのソースコードとは別に著作権が生きているため、技術解説ではエンジンのコードとアセットを明確に区別すること
- `ioquake3`をはじめ活発な派生実装が多数存在し、`OpenArena`のようにアセットも含めて完全にオープンなクローンも存在する
