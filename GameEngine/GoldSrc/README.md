## この系統について

Valve が開発したゲームエンジン。1998年の『Half-Life』で採用され、当時の FPS ゲームシーンに革新をもたらした。

**重要な特徴として、id Tech や Build Engine と異なり、GoldSrc のエンジンコア部分はソースコード公開されていない。** ただし、限定的な SDK と実行ファイルは入手可能であり、MOD 開発文化の中心となった歴史的に重要なエンジンである。

このフォルダでは以下のように分けて記録する。個別の技術詳細は各ファイルを参照:

- [`README.md`](./README.md) — このファイル(系統全体の概要)
- [`HalfLife.md`](./HalfLife.md) — Half-Life の技術解説
- [`SDKAndAccess.md`](./SDKAndAccess.md) — SDK・アクセス方法・ソースコード非公開について
