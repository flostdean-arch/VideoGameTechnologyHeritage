# ConsoleTechnology

## 概要

家庭用ゲーム機で利用されたハードウェア技術についてまとめます。

限られた性能とコストの中で、多くの革新的な技術が開発されました。

---

## 主な技術

### CPU技術

- 8bit CPU
- 16bit CPU
- 32bit CPU
- RISCアーキテクチャ
- マルチコアCPU

### グラフィックス技術

- タイル描画
- スプライト処理
- ポリゴン描画
- GPU
- シェーダー処理

### メモリ技術

- VRAM
- SRAM
- DRAM
- キャッシュメモリ
- 統合メモリアーキテクチャ

### 周辺技術

- コントローラー
- 通信機能
- 拡張機器
- センサー入力

---

## 技術的意義

家庭用ゲーム機は、専用ハードウェアによる効率化と大量生産によってゲーム産業を大きく発展させました。

---

## 目的

家庭用ゲーム機における技術革新とハードウェア設計思想を保存します。
