# History

## 概要

このセクションでは、ビデオゲーム技術の歴史を扱います。

本リポジトリでは、単に「いつ・何が登場したか」を記録するのではなく、

- なぜその技術が生まれたのか
- どのような課題を解決したのか
- 後の技術へどのような影響を与えたのか

という**技術の系譜**を重視しています。

ゲーム技術は単独で発展したものではなく、多くの技術が積み重なりながら現在のゲーム開発へと繋がっています。

その流れを後世へ残すことが、このセクションの目的です。

---

# 記述方針

各記事では以下の点を重視します。

- 技術が誕生した背景
- 当時の課題
- 採用された技術的な工夫
- 後継技術との関係
- 現代のゲーム開発への影響

単なる年表や製品紹介ではなく、「技術がどのように進化してきたのか」を理解できる資料を目指します。
