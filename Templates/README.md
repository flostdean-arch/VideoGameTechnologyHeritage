# Templates

## 概要

このディレクトリでは、本リポジトリで利用するMarkdownテンプレートを管理します。

記事の構成を統一することで、

* 記事ごとの品質を一定に保つ
* 新しい記事を作成しやすくする
* 読者が情報を探しやすくする

ことを目的としています。

各テンプレートは必要に応じて編集・拡張して構いませんが、既存の記事との整合性を考慮し、大きな変更を行う場合はリポジトリ全体への影響を確認してください。

# テンプレート利用時の方針

本リポジトリでは、単なる製品紹介ではなく**ゲーム開発技術の保存・継承**を目的としています。

そのため、記事を執筆する際は以下の点を意識してください。

* 技術的な内容を中心に記述する
* 歴史的背景を可能な限り記録する
* 一次資料や公式資料を優先して参照する
* 推測ではなく、確認できた事実を記述する
* 現在の入手性や保存状況も重要な情報として扱う

---

# 記事作成の基本方針

記事の構成はテンプレートを基本としますが、内容に応じて見出しの追加・削除を行って構いません。

ただし、

* 概要
* 歴史
* 関連項目
* 参考資料

については、可能な限り記載することを推奨します。
