# Core Technologies

## 概要

Core Technologiesでは、**特定のゲームエンジンに依存しないゲーム開発技術**を扱います。

ゲームエンジンは、それぞれ独自の設計や実装を持っていますが、その多くは共通する基礎技術の組み合わせによって成り立っています。

本ディレクトリでは、それらの技術の概要や歴史、代表的な利用例を広く紹介します。

各ゲームエンジン固有の実装や設計については、`GameEngines/` 以下の各記事を参照してください。

---

# このディレクトリで扱う内容

以下のような、ゲームエンジンを問わず利用される技術を対象とします。

* 2Dグラフィックス
* 3Dグラフィックス
* ゲームプログラミング
* AI
* 物理演算
* オーディオ
* ネットワーク
* データ構造・アルゴリズム

詳細な実装方法ではなく、「どのような技術なのか」「どのように発展してきたのか」を中心に記録します。

# GameEnginesとの違い

本リポジトリでは、ゲームエンジンと共通技術を明確に区別しています。

**Core Technologies**

* 技術そのものを解説する
* エンジンに依存しない内容を扱う
* 広く概要を理解することを目的とする

**Game Engines**

* 各ゲームエンジン固有の設計や実装を扱う
* 技術がどのように採用・発展したかを記録する
* 実際の開発環境や保存状況についても記録する

例えば、BSP（Binary Space Partitioning）は `CoreTechnologies/Graphics/` でアルゴリズムや考え方を解説し、Unreal Engineやid Techの記事では、それぞれの実装や利用方法を解説します。

---

# 本リポジトリでの位置付け

本リポジトリは、ゲーム開発技術の歴史を記録することを目的としています。

現在はゲームエンジンに重点を置いていますが、今後はゲームエンジンを支える共通技術についても順次記録を充実させていく予定です。

Core Technologiesは、そのための基盤となるディレクトリです。
