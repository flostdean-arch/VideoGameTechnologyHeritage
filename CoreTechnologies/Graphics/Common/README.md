# Common Graphics Technologies

## 概要

このディレクトリでは、2D・3Dの区別なく利用されるゲームグラフィックス技術を扱います。

これらの技術は、多くのゲームエンジンやゲームジャンルに共通する基礎技術であり、ゲーム開発全体を理解するための土台となります。
