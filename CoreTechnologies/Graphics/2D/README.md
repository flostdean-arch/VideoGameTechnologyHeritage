# 2D Graphics

## 概要

このディレクトリでは、2Dゲームで利用される描画技術を扱います。

1980年代から現在に至るまで、2Dゲームでは限られたハードウェア性能を活かすため、さまざまな描画技術が考案されてきました。

これらの技術は、現在の2Dゲーム開発にも受け継がれています。
