# 3D Graphics

## 概要

このディレクトリでは、3Dゲームで利用される描画技術や空間管理技術を扱います。

1990年代以降、リアルタイム3Dグラフィックスはゲーム開発の中心技術となり、多くのゲームエンジンでさまざまな手法が採用されてきました。

本ディレクトリでは、特定のゲームエンジンに依存しない共通技術を中心に、その歴史や基本的な仕組みを記録します。
